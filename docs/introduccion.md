# Introducción

El pádel es un deporte en constante crecimiento y muchos jugadores no disponen de una pala propia o desean probar diferentes modelos antes de comprar una.

Esta plataforma web surge como una solución para el **alquiler de palas de pádel**, permitiendo a los usuarios:
- Consultar un catálogo de palas disponibles
- Alquilar palas por un período determinado
- Visualizar características y precios
- Gestionar reservas de forma sencilla

## Objetivo
El objetivo principal del proyecto es desarrollar una aplicación web intuitiva y eficiente que facilite el alquiler de palas de pádel tanto para jugadores ocasionales como habituales.

## Público objetivo
- Jugadores de pádel amateurs
- Clubs de pádel
- Personas que desean probar diferentes palas antes de comprar
