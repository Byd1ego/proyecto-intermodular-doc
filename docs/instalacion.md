# Instalación

Esta sección describe los pasos necesarios para instalar y ejecutar la plataforma de alquiler de palas de pádel en un entorno local.

## Requisitos previos
- Node.js / Python / PHP (según el backend utilizado)
- Gestor de paquetes (npm, pip, composer, etc.)
- Base de datos (MySQL, PostgreSQL o MongoDB)
- Navegador web moderno

## Pasos de instalación

```bash
1. git clone https://github.com/usuario/alquiler-palas-padel.git
2. cd alquiler-palas-padel
3. npm install
4. npm start
