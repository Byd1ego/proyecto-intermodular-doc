# Arquitectura del Sistema

La plataforma sigue una arquitectura web moderna basada en separación de responsabilidades.

## Descripción general
El sistema se divide en:
- Frontend: Interfaz de usuario
- Backend: Lógica de negocio
- Base de datos: Gestión de información

## Componentes principales

### Frontend
- Aplicación web responsiva
- Consumo de API REST
- Visualización del catálogo y reservas

### Backend
- Gestión de usuarios
- Gestión de palas
- Control de alquileres y disponibilidad

### Base de datos
- Usuarios
- Palas
- Reservas
- Historial de alquileres
